# Dynamic Multi-Category Catalog

A responsive frontend web application built using React.js that dynamically displays products grouped by categories such as Cars, Bikes, Phones, and Computers.

Live Demo: ([Netlify Link](https://catalog-app-ui.netlify.app/))
GitHub: ([GitHub Link](https://github.com/PraveenMadiwal/catalog-app))

## Live Features

- Category-wise product sections
- Dynamic rendering using JSON data
- Product detail page
- Responsive UI for Mobile / Tablet / Desktop
- Reusable component architecture

## Approach / Implementation Idea

The project was designed using reusable React components for scalability and clean code structure.

### Home Page:
- Products are grouped dynamically based on category.
- Each category section displays preview cards.

### Detail Page:
- Clicking a product navigates to item detail page.
- The `itemprops` array is looped dynamically to render category-specific attributes.

This makes the application flexible for future categories without changing UI logic.
